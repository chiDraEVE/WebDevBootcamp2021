  6. Concat, indexOf, includes & reverse

    concat - merge arrays

    includes - look for a value

    indexOf - just like string.indexOf

    join - creates a string from an array

    reverse - reverses an array

  7. Slice & Splice

    slice - copies a portion on an array

    splice - removes/replaces elements

    sort - sorts an array

  8. Reference Types & Equality Testing

    You can't compare arrays with == or ===. You can only check the reference. When you store an array in a new variable it's not a copy, but a reference.

  9. Arrays + Const

    An array that's declared as const can't be reassigned, but the content is still mutable.
