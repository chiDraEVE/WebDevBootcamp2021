  2. Function Scope

    Variables in a function can only be called in the function itself. Be aware, that you could use the same variable name, one in the function and one global. Try to avoid that!

  3. Block Scope

    A block could be an if-statement. Everything in curly braces is a block. Be aware, that variables defined with var can be accessed anyway. Don't use var!

  4. Lexical Scope

    Inner functions have access to variables from parent functions

  7. Returning Functions

    You can generate a function in a function and return it. You could save generated functions in variables and reuse them all over the place.

  8. Defining Methods

    All methods are functions, but not all functions are methods. Methods are functions in objects and are called after a dot.
