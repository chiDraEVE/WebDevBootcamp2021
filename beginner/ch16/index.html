  6. Running JavaScript From A Script!

    Running JS Script
    You don't see variables you didn't console.log(), but you've access to them.
    Generally you place <script src="app.js"></script> at the end of the body, because you want the whole HTML loaded before using JavaScript. Else it wouldn't appear in the DOM and the site loads slower.
    A modern approach is to use async or defer. I found this StackOverflow answer when I googled why to put the script to the end: Where should I put <script> tags in HTML markup?
