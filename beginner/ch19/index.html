  11. Iterating over Objects

        const testScores = {
    keenan: 80,
    damon: 67,
    kim: 89,
    shawn: 91,
    marlon: 72,
    dwayne: 77,
    nadia: 83,
    elvira: 97,
    diedre: 81,
    vonnie: 60
}

for (let person in testScores) {
    console.log(`${person} scored ${testScores[person]}`);
}

    You can also iterate through an object converting it to an array and can access key, value or both using Object.keys(testScores), Object.values(testScores), Object.entries(testScores)
