  6. Rest Params

    It looks like spread, but it's not! Tried something myself about the arguments array: I can give a functions parameters and call them either by the parameter or the arguments array. I can also pass more arguments by calling a function as it has parameters defined and access them with the arguments array.
    Remember arguments is not really an array and you don't have array methods. Loop through all elements to either calculate what you want or store them in an actual array.
    Or you use Rest Params: function name(...parameter). Then parameter is an actual array.
