  11. Event Bubbling

    Event Bubbling is called when nested elements have many event listeners. Every listener is triggered. It's called bubbling because you can imagine bubbles rising in a fish tank. The bubble is triggering the next upper events.
    e.stopPropagation() will stop this behaviour
